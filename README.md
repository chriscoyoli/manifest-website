# Manifest Carmel Valley - Website Prototype

A redesign prototype for [Manifest](https://www.manifestcarmel.com), the eco-lux consignment
boutique at 308 Mid Valley Center, Carmel Valley, CA.

## What's in the prototype

- Brand-true design: Manifest's teal (#093C40, sampled from the logo) with a coastal,
  feminine palette (seafoam, dusty rose, shell)
- Hero with live hours, address, and the real storefront photo from manifestcarmel.com
- New arrivals grid (designed to sync from Square inventory) with brand logos and photography
- Digital consignment intake flow: account registration, drop-off slot picker, item list,
  e-sign agreement with signature pad, and handoff to the Consignor Connect portal
- Events with one-tap RSVPs (Sip & Shop Fridays), Instagram strip, email capture
- Google Map embed, social links (Instagram, Facebook, Yelp), brand favicon

## Run it

It's a single self-contained `index.html`. Open it in a browser, or serve via GitHub Pages.

## Status
